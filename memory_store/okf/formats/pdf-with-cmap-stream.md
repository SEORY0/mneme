# Schema
## Structure
Ghostscript CMap data is PostScript resource syntax embedded either as standalone resource text or as PDF stream dictionaries referenced by fonts. A usable CMap declares CIDSystemInfo, CMapName, CMapType, codespace ranges, and mapping sections such as bfchar, bfrange, cidchar, or cidrange. Source code widths and destination value string widths are distinct invariants.
