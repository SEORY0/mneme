# Html Doctype Format

## Round 27 Factual Contract

- The relevant document shape is ordinary HTML with a doctype declaration before the body.
- The doctype name may be followed by a SYSTEM literal or a PUBLIC literal plus optional system literal, with quoted identifiers parsed before document construction.
- Encoding can be inferred from an initial encoding signal or high-bit content, and conversion-buffer growth is relevant to the literal parser's closing-quote path.
